Wat is Linux?

Linux is een besturingssysteem voor computers net zoals bijv. Windows. Het zorgt ervoor dat de hardwarecomponenten, printers, etc. en de diverse softwareprogramma's doen wat u wilt

De filosofie van Linux

Linux is gebaseerd op een filosofie die zegt dat software en ideeën vrij gedeeld moeten kunnen worden. Het is vrij beschikbaar voor iedereen en iedereen mag het uitbreiden met als enige echte verplichting dat iedereen die verbetering dan weer moet delen. Dit soort software wordt "Open Source" genoemd.

Diversiteit

Linux werkt op heel veel verschillende systemen van hele oude pc's tot de servers van Google en de Supercomputer van IBM

Voor- en nadelen in vergelijking met Windows / OS X

Voordelen:

  • Linux is gratis. Linux is in tegenstelling tot Windows en Mac OS X gratis te downloaden.
  • Linux wordt veel geüpdate. Linux wordt vaak geüpdate omdat er veel programeurs over de hele wereld constant aan werken.
  • Stabiel, veilig en betrouwbaar. En makkelijk te installeren. Bij de meeste Linux distributies is het veel makkelijker om te installeren dan Windows
  • Beveiligingsproblemen worden snel ontdekt en snel opgelost. Vergelijk het bijvoorbeeld eens met een auto. Als je bij aankoop van een auto niet onder de moterkap mag kijken dan kan je later voor vervelende verassingen komen te staan. Als je open software installeert hebben al veel deskundigen onder de 'moterkap' gekeken en dit verbeterd.
  • Onafhankelijk van softwarefabrikant. Als iemand denkt dat hij iets aan Linux kan verbeteren dan heeft hij de vrijheid om dat te doen.

Nadelen:

  • Linux lijkt niet op vertouwde software. Dus als je wil beginnen met Linux moet je vaak veel nieuwe dingen leren.
  • Je moet het zelf installeren. Windows en OS X staan al voorgeinstalleerd op je computer bij linux moet je dat zelf doen.
  • Voor nieuwe/onbekende hardware zijn geen drivers beschikbaar.
  • Bepaalde documenten kunnen alleen worden geopened door de software van één fabrikant. En als die software dus niet beschikbaar is voor Linux kan dit een probleem opleveren.

Linux Distributies

Linux wordt verspreid via Linux-Distributies ook wel distro's genoemd. Een distro zorgt voor een basis van programma's voor Linux zodat je meteen aan de slag kan en niet eerst bezig bent met allemaal programma's installeren. distro's zorgen er ook voor dat alles "out of the box" meteen werkt. Vaak voegen distro's ook hun eigen thema eraan toe.

Distro's zijn vaak gericht op bepaalde taak. Zo heb je bijvoorbeeld simpele en makkelijke te gebruiken distro's zoals Ubuntu en Mint of distro's speciaal gemaakt voor servers zoals CentOS. Een opmerkelijk Linux distro is Android. Dat is het besturingssysteem van Google gemaakt voor smartphones en tablets.
